Project Description

Our Project is of Ethereum Shared wallet which is based on Web3.0 platform and made using the programming language Solidity.
This project uses the core concept of the blockchain i.e. Hashing which enables all the account transefer to be encrypted.
This is a Wallet that is controlled by the Owner only . Anyone can deposit the amount into the wallet but the withdraw facility is only limited to Owner only.
There are also function such as allowing other account as allowance, trasferring the ownership of the account, knowing owner of that account et.c.
Some of the real world of this account are:
->Allowance for Children per day/week/month to be able to spend a certain amount of funds.

->Employers give employees an allowance for their travel expenses.

->Businesses give contractors an allowance to spend a certain budget.
